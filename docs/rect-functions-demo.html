Canvas fillRect() vs strokeRect() Complete Guide

1. Basic Usage
// Basic filled rectangle ctx.fillStyle = 'blue'; ctx.fillRect(50, 50, 100, 75); // Basic outlined rectangle ctx.strokeStyle = 'red'; ctx.lineWidth = 2; ctx.strokeRect(200, 50, 100, 75);
2. Style Properties
// Different fill styles ctx.fillStyle = 'solid color'; // #ff6b6b ctx.fillStyle = 'gradient'; // Linear/Radial gradients ctx.fillStyle = 'pattern'; // Image patterns // Different stroke styles ctx.strokeStyle = 'color'; // #4ecdc4 ctx.lineWidth = thickness; // 1, 3, 5, etc. ctx.lineCap = 'butt|round|square'; // Line endings ctx.lineJoin = 'miter|round|bevel'; // Corner styles
3. Combined fillRect() + strokeRect()
// Draw filled rectangle first ctx.fillStyle = '#e74c3c'; ctx.fillRect(50, 50, 120, 80); // Then add border on top ctx.strokeStyle = '#2c3e50'; ctx.lineWidth = 3; ctx.strokeRect(50, 50, 120, 80); // This creates a filled rectangle with border!
4. Advanced Styling with Gradients
// Linear gradient const gradient = ctx.createLinearGradient(0, 0, 200, 0); gradient.addColorStop(0, '#ff6b6b'); gradient.addColorStop(1, '#4ecdc4'); ctx.fillStyle = gradient; ctx.fillRect(50, 50, 200, 80); // Radial gradient const radial = ctx.createRadialGradient(350, 90, 0, 350, 90, 60); radial.addColorStop(0, '#ffecd2'); radial.addColorStop(1, '#fcb69f'); ctx.fillStyle = radial; ctx.fillRect(300, 50, 120, 80);
5. Negative Dimensions
// Negative width - draws leftward ctx.fillStyle = '#3498db'; ctx.fillRect(200, 50, -100, 60); // x=200, but draws left 100px // Negative height - draws upward ctx.fillStyle = '#e67e22'; ctx.fillRect(50, 150, 100, -60); // y=150, but draws up 60px // Both negative - draws up and left ctx.fillStyle = '#9b59b6'; ctx.fillRect(350, 150, -80, -60);
6. Practical Examples
// Button with hover effect function drawButton(x, y, width, height, text, isHovered) { // Fill background ctx.fillStyle = isHovered ? '#0056b3' : '#007bff'; ctx.fillRect(x, y, width, height); // Add border ctx.strokeStyle = '#004085'; ctx.lineWidth = 2; ctx.strokeRect(x, y, width, height); // Add text ctx.fillStyle = 'white'; ctx.font = '16px Arial'; ctx.textAlign = 'center'; ctx.fillText(text, x + width/2, y + height/2 + 5); } // Progress bar function drawProgressBar(x, y, width, height, progress) { // Background ctx.fillStyle = '#e9ecef'; ctx.fillRect(x, y, width, height); // Progress fill ctx.fillStyle = '#28a745'; ctx.fillRect(x, y, width * progress, height); // Border ctx.strokeStyle = '#dee2e6'; ctx.strokeRect(x, y, width, height); }
Key Differences Summary:
• fillRect(): Creates solid/filled rectangles
• strokeRect(): Creates outlined/hollow rectangles
• Both use the same parameters: (x, y, width, height)
• Can be combined for filled rectangles with borders
